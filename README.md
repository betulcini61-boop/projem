# Hucre Turu

Statik, iki dilli bir hucre turu sitesi.

## Klasor yapisi

```text
.
|-- index.html
|-- pages/
|   |-- prokaryote.html
|   `-- eukaryote.html
|-- assets/
|   |-- css/
|   |   `-- styles.css
|   `-- js/
|       |-- app.js
|       |-- i18n.js
|       `-- tours.js
`-- README.md
```

## Sayfa destegi

- Ana sayfa: Prokaryot / Okaryot secimi
- Prokaryot tur sayfasi
- Okaryot tur sayfasi

Yeni tur sayfasi eklemek icin `pages/` altina yeni HTML dosyasi koyup `assets/js/tours.js` icindeki veri yapisina yeni tur ekleyebilirsin.

## Calistirma

Tamamen statiktir. `index.html` dosyasini tarayicida acman yeterli.
